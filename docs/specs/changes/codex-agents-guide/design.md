# Design: codex-agents-guide

## 总体设计

`AGENTS.md` 放在仓库根目录，作为 Codex agent 的优先工作协议。它不替代 PRD、开发规划或 vibe coding 规范，而是把这些文档转化为 agent 执行任务时必须遵守的操作规则。

## 内容结构

- 项目背景
- 重要文档
- 标准工作流
- Git 规范
- Spec 规则
- 产品约束
- 推荐架构方向
- 安全边界
- 前端、后端、AI 助手实现要求
- 测试与验收
- 三人协作方式
- Agent 回复规范
- 当前阶段优先级

## 兼容性

后续如果引入更细的子目录 `AGENTS.md`，子目录文件可以补充局部规则，但不能违反根目录规则。
