# 桌面智算 Global Spec

本文件定义桌面智算 V1.0 的系统级边界。Global Spec 优先级高于 Module Spec、OpenSpec Change 和代码实现。

## 产品定位

桌面智算是面向个人电脑、本地工作站和小型实验环境的本机 AI 训练、推理与资产管理工作台。它帮助用户在本机完成 AI 项目管理、数据导入与检查、训练任务执行、推理服务创建、模型资产沉淀、运行环境管理和 AI 辅助诊断。

## 系统边界

系统必须围绕本机工作流构建：

- 管理本地项目、数据、环境、任务、模型资产和推理服务。
- 所有核心能力必须同时考虑 UI 入口和 AI 助手入口。
- AI 助手是操作中枢，但不是稳定业务状态的唯一事实源。
- 后端负责业务 API、本地执行、任务状态、日志采集和服务进程管理。
- AI 助手负责意图识别、上下文整合、方案生成、诊断和受控工具编排。

## 一级模块

V1.0 一级模块固定为：

- 首页
- 项目
- 任务
- 数据
- 模型
- 设置

新增一级模块必须先更新 Global Spec，并由项目负责人或架构负责人确认。

## 核心数据模型

V1.0 核心实体包括：

- `Project`：项目档案，保存长期上下文和资源关系。
- `Task`：训练任务或推理服务，保存执行配置、状态、日志和产物引用。
- `Dataset`：本地数据集记录、检查结果和异常项。
- `ModelAsset`：模型文件、目录、来源任务、格式、检查结果和服务关联。
- `RuntimeEnvironment`：Python、CUDA、框架、包管理器和环境检测结果。
- `AiAssistantModel`：AI 助手模型配置，API Key 必须脱敏保存和展示。

项目不直接承接复杂执行入口；训练脚本、命令、工作目录、输出目录等复杂执行配置属于 Task。

## 状态机规则

`Task.status` 是唯一主状态，阶段进度使用 `phase` 表达。

任务主状态：

```text
等待配置
待启动
检查中
排队中
运行中
停止中
已停止
已完成
启动失败
失败
异常
```

数据集状态：

```text
未检查
检查中
可用
存在警告
不可用
```

模型资产状态：

```text
未检查
检查中
可用
存在警告
不可用
已关联推理服务
```

运行环境状态：

```text
未检测
检测中
可用
需修复
修复中
不可用
创建中
```

AI 模型配置状态：

```text
待检测
可用
异常
```

新增主状态必须先更新 Global Spec 或对应 Module Spec。

## 本地执行规则

- 所有训练、服务、环境检测、依赖安装都必须由后端受控执行器或经过审计的工具执行。
- 后端必须记录任务状态、原始日志、结构化错误、退出码和可展示建议。
- 长任务必须可追踪、可取消、可查看日志。
- 推理服务必须记录端口、进程、健康检查、endpoint 和启动失败原因。
- AI 助手不能绕过后端直接任意执行 shell。

## 文件系统安全规则

- 导入数据集、模型和环境默认只记录本地路径，不复制大文件。
- 文件扫描必须限制范围，禁止无边界扫描整盘。
- 删除本机文件、覆盖配置、清理训练产物必须明确列出影响路径并等待用户确认。
- 本地路径、用户名、日志片段在展示给 AI 或外部模型前必须按风险脱敏。

## AI 动作边界

AI 行为分为：

- `explain`：解释字段、状态、日志和错误，不改变状态。
- `field_suggestion`：生成表单或参数建议，由用户提交。
- `action_request`：请求执行真实动作，必须走后端受控工具和确认机制。

AI 不能：

- 直接执行未登记工具。
- 绕过用户确认启动训练、停止任务、删除文件、安装依赖或暴露端口。
- 明文输出 API Key、token、环境变量。
- 将本地数据集、模型权重或日志上传到外部服务，除非用户确认。

## 风险确认机制

低风险：只读信息，不改变本机状态，不需要确认。

中风险：新增或修改产品内记录，但不启动进程、不修改文件，提交前展示摘要确认。

高风险：会启动或停止进程、占用资源、修改环境、清理文件、删除记录、暴露端口或覆盖配置，必须弹出高风险确认。

AI 发起的动作必须遵守同一分级。

## V1.0 非目标

以下能力不在 V1.0 范围：

- 云端训练资源调度。
- 多机多卡分布式训练。
- 数据标注平台。
- 复杂数据清洗流水线。
- 团队协作权限。
- Kubernetes 部署。
- 大模型预训练。
- RLHF / DPO。
- 完整 Git 仓库管理、代码 diff、commit、branch 产品能力。
- 无用户确认的自动训练、自动删除文件、自动安装未知依赖。
