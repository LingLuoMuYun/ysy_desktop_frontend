# Agent Module Spec

本文件定义桌面智算 AI 助手与智能编排边界。当前仓库 AI 助手位于 `agent/`。

## AI 助手角色

AI 助手是项目、环境、训练、服务、诊断和分析的操作中枢，不是普通聊天框。它负责：

- 识别用户意图。
- 读取当前页面、当前对象和当前项目上下文。
- 解释字段、状态、日志和错误。
- 生成表单建议和执行计划。
- 通过后端受控工具发起可确认动作。
- 输出失败原因、证据和下一步建议。

AI 助手不承接稳定业务状态的唯一事实源。

## 行为类型

`explain`：

- 解释字段、状态、错误、日志和概念。
- 不改变产品记录、文件、进程或环境。

`field_suggestion`：

- 生成表单字段、参数、修复方案或配置建议。
- 用户确认提交后才写入。

`action_request`：

- 请求创建项目、检查环境、创建任务、启动训练、启动服务、停止任务等真实动作。
- 必须调用后端受控工具。
- 高风险动作必须等待用户确认。

## 上下文读取范围

AI 可以读取：

- 当前页面模块和选中对象摘要。
- Project、Task、Dataset、ModelAsset、RuntimeEnvironment 的结构化字段。
- 任务日志片段和错误摘要。
- API 返回的结构化诊断结果。
- 用户主动附加的本地文件摘要。

AI 默认不读取：

- 未经用户选择的大型数据集内容。
- 模型权重文件内容。
- 明文 API Key、token、环境变量。
- 与当前任务无关的本机目录。

## 禁止行为

AI 不能：

- 绕过后端直接执行 shell。
- 自动删除文件、覆盖配置、安装未知依赖、暴露端口或启动长时间训练。
- 将本地路径、日志、数据集或模型内容发送到外部服务，除非用户确认。
- 明文展示 API Key、token、环境变量。
- 把生成建议伪装成已经执行的结果。
- 在缺少必要参数时编造路径、端口、模型或环境状态。

## 脱敏规则

输出给用户或外部模型前必须脱敏：

- API Key、token、secret。
- 环境变量值。
- 用户主目录、企业目录、敏感项目路径。
- 日志中的凭据、URL query secret、Authorization 头。

## 高风险动作确认

AI 发起高风险动作时必须输出结构化执行计划：

```text
任务：启动训练任务
步骤：
1. 检查任务配置
2. 检查运行环境
3. 检查数据集
4. 检查资源
5. 等待用户确认
6. 调用后端启动任务
```

确认前不得执行真实动作。确认后仍必须通过后端受控工具执行。

## 工具契约

每个 AI 工具必须满足：

- 输入参数明确。
- 输出结果结构化。
- 失败原因可解释。
- 日志可追踪。
- 风险等级明确。
- 高风险步骤可中断或确认。
