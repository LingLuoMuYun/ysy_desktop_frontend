# 侧边栏软件图标

## 背景

左侧导航栏顶部品牌区域需要替换为用户提供的 SVG 软件图标，并支持点击图标直接返回首页。

## 目标

- 在左侧导航栏顶部品牌区域展示提供的 SVG 软件图标。
- 点击软件图标后切换到首页。
- Electron 窗口创建时使用同一 SVG 作为窗口图标资源。
- 图标资源进入前端渲染资源和 Electron resources 目录，便于 UI 与桌面壳复用。
- 打包文件列表包含 Electron 图标资源，保证生产运行时可访问。

## 非目标

- 不调整顶部栏布局。
- 不重做侧边栏导航结构。
- 不替换 Windows 打包所需的 `.ico` 安装包图标格式。
- 不新增路由或后端能力。

## 风险等级

低风险。仅调整前端导航入口、静态资源和 Electron 窗口图标配置。

## 验收标准

- 左侧导航栏顶部品牌区域显示提供的 SVG 图标。
- 在非首页页面点击该图标可回到首页。
- TypeScript 类型检查通过。
