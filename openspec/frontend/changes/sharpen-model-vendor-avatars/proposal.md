# 模型厂商头像清晰度优化

## 背景

模型管理中的厂商头像使用 2x 图标资源，源文件为 32×32 像素，但当前样式将图片拉伸到 46×46 显示，导致头像观感发糊。用户同时提供了 1x 图标资源，经检查 1x 为 16×16，不适合替代 2x。

## 目标

- 继续使用 2x 厂商图标资源。
- 图片头像在 46px 容器中按 32×32 原始尺寸居中显示。
- 避免浏览器放大位图导致头像模糊。

## 非目标

- 不替换厂商图标文件。
- 不调整模型卡片布局结构。
- 不新增后端字段或接口。

## 验收标准

- 厂商头像不再被拉伸到 46×46。
- TypeScript 类型检查通过。
