# AI 对话附件格式图标

## 背景

首页 AI 对话上传附件后，附件标签左侧目前只显示通用链接图标，无法根据文件格式提供明确视觉识别。用户希望选择 `windows安装指南.md` 这类 Markdown 文件后，在附件左侧显示对应的 SVG 格式图标，并优化标签样式。

## 目标

- 首页 AI 对话输入区已选附件左侧显示文件格式 SVG 图标。
- 已发送消息中的附件标签保持同样的格式图标展示。
- `.md` / `.markdown` 文件使用 Markdown 专属图标样式。
- 未识别格式使用通用文件 SVG 图标兜底。

## 非目标

- 不改变附件选择、打开、删除和发送流程。
- 不新增后端接口或 IPC 通道。
- 不读取附件内容或上传本地文件内容。

## 影响模块

- 前端：首页 AI 对话附件展示。

## 风险等级

低风险。仅调整前端展示，不改变本机文件状态。

## 验收标准

- 选择 `.md` 文件后，附件名称左侧显示 Markdown SVG 图标。
- 附件名称超长时仍保持单行省略，不撑开对话框。
- 未识别扩展名仍显示通用 SVG 文件图标。
- TypeScript 类型检查通过，或记录未通过原因。
