# Add Project Detail Dialog

## 背景

用户要求项目列表中点击项目后弹出项目详情，并按提供截图还原详情弹窗的视觉与信息结构。

## 目标

- 项目卡片可打开项目详情弹窗。
- 弹窗展示项目标题、状态、项目概览 / 操作记录 / 任务记录 / 模型资产四个 Tab。
- 项目概览展示项目名称、项目类型、项目状态、项目保存位置、默认运行环境和更新时间。
- 底部提供“创建部署任务”和“创建训练任务”入口占位。
- 梳理当前项目页前端逻辑，形成项目分析文档和前端经验报告，便于后续接入详情子列表和真实任务入口。

## 非目标

- 不新增后端接口。
- 不在本次接入操作记录、任务记录、模型资产列表的真实数据。
- 不启动真实训练任务或部署任务。
- 不把本次文档中的待确认事项视为已确认需求。

## 验收标准

- 点击项目名称或项目卡片空白区域打开项目详情弹窗。
- 弹窗整体尺寸、头部、Tab、信息卡片和底部按钮贴近参考图。
- 项目概览字段来自项目详情接口或列表 fallback。
- 非概览 Tab 展示明确空态，不伪造接口数据。
- README、OpenSpec、AGENTS 和项目分析文档同步说明当前项目页结构、项目详情边界和待确认项。
